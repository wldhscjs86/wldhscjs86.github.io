## 1. 메커니즘

[도전 과제]

1. 사막 지역을 돌파하여 황금 스카라베를 얻어라
2. 정글 지역을 돌파하여 황금 원숭이 상을 얻어라
3. 지하 동굴을 돌파하여 시환제의 관에 도달해라
4. 모험중 자잘한 보물들을 얻어서 점수를 얻어라
5. 아이템들을 사용하여 막힌지형들을 타파하라

<br>

[재미 요소]

1. 인디아나 존스를 모티브로한 캐릭터의 컨셉
2. 아이템과 적절한 판단으로 함정들을 이겨내기
3. 주변에 떨어져있는 작은 보물들을 주워 점수경쟁
4. 인디아나 존스의 사막 결투 장면을 패러디 한 이스터에그

<br>

[키조작]

WASD  : 캐릭터 이동

SPACE : 점프

J : 공격

K : 아이템 사용

## 3. 미적요소


캐릭터 디자인 : 인디아나 존스와 흡사한 도트 디자인

몬스터 디자인 : 미라, 벌레, 뱀, 원시인, 병마용, 박쥐, 뼈무더기 

메인(인트로) 디자인 : 한 청년이 비행기에서 뛰어내릴 준비를 하고 엄지를 치켜 새운 뒤 스타트 버튼을 누르면 낙하 후 착지 사막의 모험 시작 설정 터치시 비행기 안으로 이동하며 지도를 펼치는 행동을 하고 설정화면이 나옴

인게임 디자인 : 사막, 정글, 지하미궁

아이템 디자인 : 밧줄, 권총, 다이너마이트, 삽

[음향]

메인(인트로)에 있는 버튼 클릭 시 : 스타트 버튼은 게임 제목인 we'll be rich를 외치고 하강 설정 터치 시 부스럭 거리는 소리 출력

인게임 효과음 : 지상에서는 인트로와 유사한 배경음, 지하로 진입시 약간 빠르게 진행, 보물을 얻고 돌아올때는 긴박한 배경을으로 바뀜

# [게임의 규칙] <a name='11'> </a>

### 1) 핵심 규칙
스테이지를 진행하며 함정을 피하고 몬스터를 죽이며 보스 몬스터를 처치하고 보물을 얻어라.<br>

### 2) 보조 규칙
캐릭터가 몬스터에게 피해를 입거나 함정에 닿아 체력이 0이 되면 사망한다.<br>
캐릭터가 사망 시 안전지대로 플레이어를 이동된다.<br>

### 3) 게임에서 사용될 공식
모든 캐릭터가 피해를 입어 체력이 0이 될 시 사망한다.<br>
함정 발동영역에 캐릭터가 접촉할 시 함정이 발동된다.<br>
캐릭터가 Item (Cola)를 사용할 때 채력을 2 회복한다.<br>
캐릭터가 Item (Pot)을 주워 던졌을 시 안에서 Item(Pot)을 제외한 소모성 아이템이 일정 확률로 나온다.<br>
캐릭터가 Item (Dynamite)를 던졌을 시 일정한 범위의 오브젝트들이 전부 파괴된다.<br>
캐릭터가 Item을 소지한 상태에서 J 버튼을 누를 시 해당 아이템을 내려놓는다.<br>
캐릭터가 Item을 소지한 상태에서 이동 버튼을 누른 상태에서 J 버튼을 누를 시 해당 아이템을 이동 버튼의 방향으로 던진다.<br>
캐릭터가 모든 행동 상태에서 L버튼을 누를 시 캐릭터 위치에서 위로 밧줄 앵커를 발사하여 앵커로부터 6블록 길이의 오르내릴 수 있는 밧줄을 생성한다.<br>
캐릭터가 빈손인 상태에서 아이템을 사용 시 소지품의 Item (Dynamite)를 손에 든다.<br>
NPC (Shopkeeper Bob)은 충돌판정을 받을 시 플레이어를 적대하게 된다.<br>
NPC (Shopkeeper Bob)은 상점지역에서 4가지의 무작위 소모품 아이템을 일정 가격으로 판매를 한다.<br>

# [요구사항] <a name='12'> </a>

## 프로그래밍 요구사항

### [메뉴]

1. 시작화면, 배경화면이 있다.
2. 시작화면에는 새로하기, 이어하기, 옵션, 게임종료 버튼이 있다.
3. 종료를 클릭 시 게임이 종료 된다.
4. 옵션을 클릭 시 BGM과 SE의 소리를 설정 할 수 있다.

<br>

### [스테이지 구성]

1. 2D 사이드 뷰 시점 카메라를 사용한다.
2. 게임 시작 시 안전지대로 이동되게 한다.
3. 안전지대에는 NPC (Aviator jimmy)와 스테이지 진입 입구를 만든다.
4. NPC (Aviator jimmy)는 클리어 한 스테이지가 없을 시 힘내라는 대화를 출력한다.
5. NPC (Aviator jimmy)는 클리어 한 스테이지로 이동 시켜주게 한다.
6. 스테이지 진입 입구에서 W 버튼을 눌러 스테이지에 진입할 수 있게한다.
7. 스테이지 안에는 특정한 위치에 상점 지역이 생성하게 한다.
8. 상점 지역에는 NPC (Shopkeeper Bob)이 출현하게 한다.
9. 상점 지역에는 무작위 4가지의 소모성 아이템이 출현하게 한다.
10. 함정의 발동 조건을 충족 시 함정이 발동하게 한다..
11. 발사형 함정은 함정의 발동 조건 충족되었을 때 1회만 발동하게 한다.
12. 발사형 함정은 모든 오브젝트에 반응하게 한다.
13. 벌레 함정은 발동 조건을 충족 시 안에서 벌레 몬스터가 3마리가 소환되게 한다.
14. 점수이자 게임 내 상점에서 사용할 수 있는 재화 아이템을 만든다.
15. 한 스테이지에서 안전지대 보스룸을 제외한 3가지 바리에이션의 맵을 제작한다.

<br>

### [플레이어]

1. 게임화면 좌측 상단에 체력, 다이너마이트, 밧줄의 UI를 출력한다
2. 게임화면 우측 상단에 돈, 시간, 현제 스테이지의 UI를 출력한다.
3. 캐릭터의 이동을 W,A,S,D로 한다.
4. 캐릭터의 점프는 space로 한다.
5. 캐릭터의 공격을 J로 한다.
6. 캐릭터의 아이템 사용을 K로 한다.
7. 캐릭터의 밧줄 사용을 L로 한다.
8. 캐릭터가 아이템을 들은 상태에서 J를 눌렀을 때 아이템을 내려놓는다.
9. 캐릭터가 아이템을 들은 상태에서 이동 키와 J 키를 같이 눌렀을 때 이동 키를 누른 방향으로 던진다.
10.  모든 아이템은 던졌을 때 포물선을 그리며 떨어진다.
11. 캐릭터가 빈손에서 K 버튼을 누를 시 소지품에서 다이너마이트를 꺼낸다.
12. 다이너마이트는 소지품에서 꺼낸 순간부터 5초 후에 폭발하게 한다.
13. 다이너마이트는 폭발 시 일정반경의 오브젝트들을 전부 파괴하게 한다.
14. 다이너마이트는 폭발 시 모든 몬스터에게 3의 데미지를 준다.
15. 체력이 0이 되었을 때 사망모션 후 점수판이 나오게 한다.
16. 점수판에서 재시작과 게임 종료버튼을 만든다.

<br>

### [몬스터]

1. 몬스터는 일반 몬스터와 보스몬스터로 나눈다.
2. 모든 일반 몬스터는 체력을 1로 고정한다.
3. 모든 일반 몬스터는 머리를 밟았을 때 사망하게 한다.
4. 모든 일반 몬스터는 충돌 판정 시 캐릭터는 1의 데미지를 입는다.
5. 보스 몬스터는 15의 체력을 가진다.
6. 보스 몬스터를 한 패턴에 3의 데미지만 입게 한다.
7. 모든 몬스터는 다이너마이트에 3의 데미지를 입는다. 

<br>

### [AI]

1. 일반 몬스터에게 이동가능 area를 설정하여 설정한 area의 영역만 배회하게 만든다.
2. 플레이어를 발견 시 플레이어에게 다가가 공격하게 한다.
3. 보스 몬스터는 체력에 따라 모습이 바뀌게한다.
4. 보스 몬스터는 각기 다른 패턴을 만든다.

<br>

### [상호작용]

1. 스테이지 진입 입구판정 안에 도달 후 W를 눌러 들어갈 수 있게 설정한다.
2. NPC (Aviator jimmy)는 일정 거리로 다가가면 space 버튼으로 상호작용하게 설정한다.

<br>

### [세이브/로드]

1. NPC (Aviator jimmy)와 상호작용 시 저장되게 한다.
2. 메인화면에서 이어하기를 하면 세이브한 시점에서 안전지대에서 시작하게 만든다.

<br>

## 그래픽 디자인

### [메인메뉴]

1. 타이틀을 제작한다.
2. 처음 시작, 이어하기, 설정 버튼을 제작한다.
3. 메인화면을 제작한다.

<br>

### [스테이지]

1. 각 스테이지에 맞는 지형을 제작한다.

<br>

### [오브젝트]

1. 점수 아이템, 모든 아이템, 함정 몬스터 등 오브젝트 리소스들을 제작한다.
