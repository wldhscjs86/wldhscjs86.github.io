# 데린쿠유

이동키<br>
W : 앞으로 이동<br>
A : 좌측으로 이동<br>
S : 뒤로 이동<br>
D : 우측으로 이동<br>

많은 게임에서 사용되는 이동 키입니다. 해당 움직임은 정확히 구현 완료하였고 각각의 이동에 부드러운 모션을 삽입하여 각 방향으로 자유롭게 이동하면서 어색함이 느껴지지 않게 하였습니다. 기본적인 카메라의 움직임은 마우스를 따라가며 캐릭터가 이동하며 시점 이동이 편한 속도로 설정하였습니다.<br>

Shift 키<br>

WASD 조작 중 Shift 누르고있기 : 달리기<br>
WASD 조작 중 Shift 한번 누르기 : 구르기<br>
정지 상태 중 Shift 누르기 : 백스탭<br>

해당 키의 배정은 RPG 또는 FPS에서 익숙하게 사용하고 있는 키로 배정하였습니다. 달리기는 특별한 기능은 없지만 앞으로 구르기에는 구르는 순간 약간의 무적 판정을 두어 초보자들이 쉽게 적의 공격을 피할 수 있게 만들어 둔 기능입니다. 뒤로 백스탭은 무적 상태에 들어가지는 않지만 적은 스테미나 소모로 공격을 피할 수 있습니다.하지만 해당 기능들은 지금 동작이 불가합니다. 빠르게 스크립트를 고쳐 다시 동작하겠습니다.<br>

공격키<br>
E : 약공격<br>
E -> R : 2연속 공격<br>
R : 강 공격<br>

현재 E 와 R키는 공격버튼으로 제작하였고 해당 조작은 추후 마우스의 좌 클릭에 전부 넣을 예정입니다. 약 공격 1 은 클릭 한번, 2연속 공격은 클릭 2번, 강 공격은 클릭을 오래 누르면 나가는 방향으로 수정 예정입니다. 해당 키는 정상적으로 동작하지만 E키 또는 R키를 빠르게 연속적으로 누르면 공격이 끝나지 않은 채 모션이 캔슬 되어 수정 예정이며 2연속 공격은 약 공격_02 모션이 아닌 강 공격 모션으로 나가 해당 스크립트도 수정 예정입니다.<br>

반격기<br>
Q : 패링<br>

해당 기능은 굴러서 피하는 기능과 다른 적의 공격을 튕겨내는 반격기로 Q 키에 들어가 예정입니다. 스크립트와 애니메이션은 준비되었지만 동작이 안되어 추후 스크립트를 고쳐 나가면서 구현해나갈 예정입니다.<br>

현재 무기 구현<br>

방패 :60% - 손잡이 부분, 색 미구현<br>
메이스 : 80% - 색 미구현<br>
도끼 : 80% - 색 미구현<br>
검 : 80% - 색 미구현<br>

현재 무기는 Z 브러쉬를 사용하여 방패, 메이스, 도끼, 검까지 구현했습니다. 지금 캐릭터에게 장착 되어있는 쌍검 중 좌측의 검은 방패로 바뀔 예정이며 방패의 가죽 손잡이만 마저 구현되면 바로 장착 예정이며 이어 방패로 반격까지 이어집니다. 각각의 색은 아직 미정이며 가장 우선순위가 높은 대상입니다.

검이 캐릭터의 몸에 박혀있던 현상은 적대 몬스터를 삭제하며 해결되었으며 각각의 스크립트들을 조율하며 빠르게 다시 전투가 가능한 상태로 되돌릴 예정입니다.<br>
